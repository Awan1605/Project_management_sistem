# Project_management_sistem
Arva - Aplikasi manajemen proyek Kanban berbasis Django dengan fitur task management, drag-and-drop, dan real-time updates. Dilengkapi AI multi-provider (Gemini, OpenAI, Ollama, DeepSeek) untuk analisis prioritas tugas, AI chat assistant, dan AI developer. Mendukung subproyek, role-based access, serta tema terang/gelap.
